# Product Ideas / 产品玩法构想

## Positioning / 产品定位

### English

Pixel Agents should not feel like "a prettier terminal viewer". It should feel like an operating system for AI teammates — a place where real work and playful management overlap.

The strongest product direction is:

- Make agents feel like teammates, not processes
- Make work visible, not hidden in terminals
- Make coordination fun, not operationally heavy
- Make real output turn into visual feedback and progression

### 中文

Pixel Agents 不应该只是“更好看的终端面板”，而应该像一个给 AI 队友使用的操作系统：真实工作在这里发生，同时又带有轻度经营和游戏化体验。

最值得坚持的产品方向是：

- 让 agent 更像队友，而不是进程
- 让工作过程可视化，而不是藏在终端里
- 让协作变得有趣，而不是只有调度负担
- 让真实产出转化为可见反馈和成长奖励

## Core Loop / 核心循环

### English

A strong core loop for Pixel Agents:

1. Create or restore agents
2. Assign them tasks or working directories
3. Watch them collaborate, wait, ask, and finish
4. Intervene when needed
5. Turn completed work into visible progress in the office
6. Improve the team and office over time

### 中文

我建议的核心循环：

1. 创建或恢复多个 agent
2. 给他们分配任务或工作目录
3. 看他们协作、等待、请求、完成工作
4. 在关键节点由用户介入
5. 把完成的真实工作转化为办公室中的可见成果
6. 持续优化团队配置和办公室布局

## High-Value Features / 高价值玩法

### 1. Role Cards / 岗位卡

- English: Let each agent have a role such as product, frontend, backend, designer, QA, or reviewer.
- 中文：让每个 agent 都有岗位身份，比如产品、前端、后端、设计、测试、审查者。
- English: Roles influence visuals, labels, default prompts, and desk props.
- 中文：岗位影响角色外观、标签、默认行为和工位道具。

### 2. Task Wall / 任务墙

- English: Add a kanban-style wall where users can drag tasks onto agents.
- 中文：增加一面像看板一样的任务墙，用户可以把任务拖拽给 agent。
- English: This makes the office feel managed rather than merely observed.
- 中文：这样办公室就从“被观察”变成“被经营”。

### 3. Sub-Agent Errands / 子代理跑腿

- English: Turn sub-agents into visible short-lived helpers that branch off and return results.
- 中文：把 sub-agent 做成临时跑腿角色，出去执行子任务，再带着结果回来。
- English: This is one of the clearest visual differentiators for multi-agent workflows.
- 中文：这是多智能体协作里最有辨识度的可视化亮点之一。

### 4. Outcome Visualization / 成果具象化

- English: Completed work should leave visible traces in the office: reports, green lights, trophies, boards, or props.
- 中文：已完成的工作应该在办公室里留下痕迹，比如报告、绿灯、奖杯、公告板、道具变化。
- English: Real output should feel tangible.
- 中文：真实产出要有“摸得着”的感觉。

### 5. Agent Memory and Relationships / 角色记忆与关系

- English: Agents should build identity over time through names, specialties, and collaboration history.
- 中文：agent 应该随着时间拥有身份感，比如名字、专长、协作历史。
- English: Users should remember who is best at which kind of work.
- 中文：用户会逐渐记住“谁最擅长做什么”。

### 6. Mood and Energy / 情绪与精力

- English: Long-running or blocked sessions can shift the mood of an agent.
- 中文：长时间运行、阻塞、等待审批的 session 可以影响 agent 的状态和情绪。
- English: Small office interactions like coffee, breaks, or meetings can restore rhythm.
- 中文：咖啡机、休息区、会议桌等轻交互可以恢复节奏感。

### 7. Unlockable Office Progression / 可解锁办公室成长

- English: Reward real productivity with furniture, skins, props, and decorative upgrades.
- 中文：把真实生产成果转化为家具、皮肤、装饰、道具等奖励。
- English: The office becomes a record of accumulated work.
- 中文：办公室本身会变成工作成果的展示空间。

### 8. Daily Report / 每日战报

- English: Generate a daily summary of who worked most, who waited most, and what got done.
- 中文：每天生成一张战报，展示谁最忙、谁等待最多、今天完成了什么。
- English: This has both retention value and sharing value.
- 中文：这同时具备留存价值和传播价值。

## Suggested Priorities / 建议优先级

### P0

- Role cards / 岗位卡
- Task wall / 任务墙
- Status performance events / 状态演出
- Daily report / 每日战报

### P1

- Sub-agent errands / 子代理跑腿
- Outcome visualization / 成果具象化
- Unlockable office progression / 可解锁办公室成长

### P2

- Mood and energy / 情绪与精力
- Agent relationships / 角色关系
- Richer management systems / 更丰富的经营系统

## Product Principle / 产品原则

### English

The best version of Pixel Agents is not a game skin on top of AI tooling. It is a real AI workbench whose usability becomes more engaging because it is playful.

### 中文

Pixel Agents 最好的形态，不是在 AI 工具外面套一层游戏皮，而是把真实可用的 AI 工作台，做成一个更有参与感、更有经营感、更让人上瘾的产品。
